      About MindScan
      A Python-powered mental wellness tool built for students, by students.

      MindScan uses a Flask backend running scikit-learn's Logistic Regression model. User responses are stored in a SQLite database and used to continuously improve the model. The model is retrained automatically after each submission, so it gets smarter as more students use it.

      Technologies Used

      Python 3 Flask scikit-learn NumPy SQLite MinMaxScaler LBFGS Solver HTML/CSS/JS

      📊 Dataset & Pipeline

      The backend generates 780 synthetic student samples based on the Perceived Stress Scale (PSS-10). Real user responses are stored in SQLite and merged into the training set. Data is cleaned (outlier removal), Min-Max normalized, shuffled, and split 80/20 before training. Metrics (accuracy, precision, recall, F1) are computed on the held-out test set using sklearn's reporting tools.
